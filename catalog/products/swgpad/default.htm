Microsoft® SideWinder® game pad
Absolute Victory—Total Control. The SideWinder game pad has 6 action buttons, an 8-way cross key-pad and 2 trigger buttons.

The SideWinder Game Pad brings a new level of control to PC gaming for absolute victory and total control. The SideWinder Game Pad is a game controller for the PC with six buttons, an eight-way cross key pad, and two trigger buttons. It comes with software that allows players to map multi-button game moves or keyboard strokes to single buttons on the pad.


Microsoft SideWinder Web Site
  • Up to four SideWinder game pads can be connected simultaneously while maintaining full functionality in all units for head-to-head or same team play.
  • Map complicated multi-button combo moves into easy single button moves.
  • The pad is designed to be comfortable to hold.
